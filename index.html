    SDR with Haskell via BladeRF

    This is a Haskell binding to the libBladeRF C library. The intent here is to allow developement of Software Defined Radio (SDR) applications in our favorite functional prgramming language.

    Example programs

    There are some example programs that interact with the BladeRF SDR written in Haskell that make use of this binding in the examples/ directory.
